# Brief del proyecto

## Nombre

Recticar Presupuestos

## Objetivo

Centralizar la creación, consulta y seguimiento de presupuestos.

## Preguntas a definir

- Quiénes van a usar el sistema
- Qué datos necesita cada presupuesto
- Qué estados va a tener un presupuesto
- Si habrá exportación a PDF, impresión o envío por WhatsApp/email
- Si el sistema será web, escritorio o móvil
